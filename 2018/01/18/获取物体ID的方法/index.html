获取物体ID的方法

发表于 2018-01-18

在一般的脚本中，想要获取一个物体，常用的方法是用物体名称来选择，比如select $Box001 但是当场景中有同名的文件时，该方法就不起用，今天在老外论坛上看到一个方法，可以得到一个唯一的物体ID，方法如下：

id = $Box01.handle

该方法得到一个数字，这个数字个人理解应该是类似于整个场景集合中的下标一样的东西。

根据官方的解释，使用一下方法会更加安全：

id = $box01.inode.handle

因为如果一个物体如果有handle属性的话， $Box01.handle 获取的就是handle属性而不是想要的下标，原文如下：

This is to prevent cases where the baseobject has property called handle, like a teapot, and you get that object’s handle instead of the node handle.
